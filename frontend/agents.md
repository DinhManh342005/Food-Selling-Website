# FoodSellingWeb Frontend Design Rules

## Technology

Use only:

* HTML5
* CSS3
* Vanilla JavaScript
* Tailwind CSS

Do not use:

* React
* Vue
* Angular
* Bootstrap
* UI libraries

## Design References

Learn from:

* Tiki
* Shopee
* Lazada
* Winmart

## Anti AI Generated UI

Never create:

* Giant hero sections
* Huge empty spaces
* Floating cards
* Excessive shadows
* Portfolio style layouts
* Startup landing page layouts

Note:
* Subtle gradients and light glassmorphism are allowed sparingly to enhance visual appeal, but should not overwhelm the UI.

## Layout Principles

Prioritize:

* Product visibility
* Fast navigation
* Dense information
* Real ecommerce UX

Use:

* Fixed width container
* Category sidebar
* Product grids
* Compact spacing

Avoid:

* Oversized padding
* Full-screen sections
* Decorative elements without purpose

## CSS Rules

Border radius:

4px - 8px

Shadow:

Very subtle only

Typography:

Simple and readable

Spacing:

Tight and practical

Hover effects:

Small and professional

Animations:

Minimal

## Product Card

Contains:

* Product image
* Product name
* Region badge
* Price
* Add to cart button

The card should resemble a real ecommerce website.

Not a Dribbble concept.

## Evaluation Checklist

Before generating code, check:

1. Does this look like a real ecommerce website?
2. Would Tiki or Shopee use this layout?
3. Is there unnecessary whitespace?
4. Is the design trying too hard to look modern?
5. Does the UI prioritize products over decoration?

If any answer is problematic, redesign before generating.
